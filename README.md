## Projeto Loja

### Descrição

Esse projeto é uma atividade para o curso de Fundamentos da Computação BICT UFMA.
O mesmo faz uma simulação de um carrinho de compras em uma loja, é possivel escolher diferente opções de itens e até aplicar disconto em porcentagem, e ao final mostrar o total a ser pago.

### Como dar Run:

instalar python
botar no terminal

```shell
py main.py
```

### Exemplo de uso:

Ao dar Run no programa uma lista irá aparecer, para escolher uma opção é só escrever o número da opção mostrada na esquerda do texto e apertar Enter. Se for um produto ele será adcionado ao carrinho, a opção de adcionar disconto, irá pedir uma porcentagem quer será subtraída do total futuro, e a ultima mostra o total do carrinho e encerra o programa.
